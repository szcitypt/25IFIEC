# 演示脚本

## 演示目标

展示「瞬忆 · AI 记忆助手」从硬件触发、后端整理到手机 APP 展示的完整闭环。

## 推荐演示流程

1. 打开手机 APP，展示首页和记忆列表。
2. 展示硬件端状态，说明 ESP32 已连接 Wi-Fi。
3. 按下或触发 ESP32 端演示采集动作。
4. ESP32 向后端上传一段测试文本。
5. 后端返回 mock AI 会议整理结果。
6. APP 刷新或请求后端接口，展示会议摘要、待办事项、关键词和时间线。
7. 说明后续可替换真实语音识别、真实音频上传和真实大模型接口。

## 演示话术要点

- 当前比赛阶段优先保证演示闭环稳定。
- 硬件端已经预留 INMP441 麦克风接入位置。
- 当前不做后台录音，不做静默监听，不申请 APP 麦克风权限。
- mock 数据用于展示产品流程和交互效果，后续可以替换为真实 AI 服务。

## 兜底方案

- 如果现场网络不稳定，APP 可展示本地 mock 数据。
- 如果硬件上传失败，可由后端 mock 接口直接返回固定结果。
- 如果 Android APK 暂未打包完成，可用本地网页或开发模式展示交互。
