# 硬件接线说明

## 硬件范围

当前硬件端使用 ESP32 + INMP441 麦克风，项目代码必须放在：

```text
hardware/shunyi-esp32/
```

不要把本项目代码写入 ESP-IDF 官方安装目录。

## INMP441 常见引脚

| INMP441 | ESP32 示例引脚 | 说明 |
| --- | --- | --- |
| VCC | 3V3 | 供电 |
| GND | GND | 地 |
| SCK | GPIO14 | I2S BCLK |
| WS | GPIO15 | I2S LRCLK |
| SD | GPIO32 | I2S DATA |
| L/R | GND | 选择声道 |

实际引脚可按开发板调整，但需要同步修改 ESP-IDF 配置。

硬件端只响应明确触发动作，不做静默采集和后台监听。
